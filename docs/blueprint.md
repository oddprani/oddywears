# **App Name**: OddyWears

## Core Features:

- Product Display: Showcase available t-shirts with images, descriptions, and prices.
- Category Browsing: Allow users to browse t-shirts by category: Cars, Anime, Music.
- Hero Section: Display banners promoting new products and special offers in a visually appealing hero section.
- Shopping Cart: Enable users to add selected t-shirts to a shopping cart.
- Checkout Process: Provide a secure and straightforward checkout process.
- Order Management: Ability to see their previous orders.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey a sense of trust and reliability.
- Background color: Light gray (#F5F5F5) to provide a clean and modern backdrop.
- Accent color: Vibrant orange (#FF9800) to highlight promotions and calls to action.
- Font pairing: 'Poppins' (sans-serif) for headings and 'PT Sans' (sans-serif) for body text to maintain readability.
- Use clean, modern icons to represent product features and categories.
- Implement a responsive layout that adapts seamlessly to different screen sizes.
- Incorporate subtle animations on button hovers and image transitions to enhance user experience.
